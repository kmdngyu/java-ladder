# java-ladder
사다리타기 미션 저장소

####기능구현
- [ ] 사람이름 입력 받기 ( 쉼표로 구분 )
    - [ ] 5자 넘어가면 재입력
    - [ ] 2명 이상 입력
- [ ] 최대 사다리 높이 입력
    - [ ] 1 이상 
- [ ] 사다리 로직 구현
    - [ ] 가로라인이 겹치지 않게
- [ ] 사다리 출력
